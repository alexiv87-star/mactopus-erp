Вход по PIN

Мобильный вход без Supabase Auth. При первом входе задай PIN, далее используй его на этом устройстве.
PIN хранится только на устройстве. Supabase используется как база данных через anon key.

Отправка покупателю

Отправка не создаёт выручку. Деньги фиксируются только после кнопки «Оплата получена».

Закупка

Закупка — это партия товара. Товары можно привязать к ней в карточке товара. Cash уменьшается на сумму всех закупок.

Финансовая операция

Расходы, реклама, комиссии, упаковка и доставка за свой счёт автоматически уменьшают Cash.
